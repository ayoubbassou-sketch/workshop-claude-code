Create the import-contacts skill
Skill

Handles CSV parsing, column detection (FR + EN aliases), and batched contact import into Brevo.

Prompt
Create a skill import-contacts for the CRM-setup-expert plugin.

It parses a contacts CSV, auto-detects columns (email, firstname,
lastname, phone, company — support French and English aliases),
maps them to Brevo fields (email, FIRSTNAME, LASTNAME, SMS,
WHATSAPP, COMPANY), and batch-imports 500 contacts per call
skills/import-contacts/SKILL.md
